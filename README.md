# 🎮 KBC Advance Game

A console-based quiz game developed using Python. The game is inspired by Kaun Banega Crorepati (KBC) and includes multiple-choice questions, prize money, timer, and lifelines.

## Features

- Multiple Choice Questions
- Prize Money System
- 50-50 Lifeline
- Audience Poll Lifeline
- 10 Second Timer
- Safe Level Prize
- Random Question Order

## Technologies Used

- Python 3

## How to Run

1. Install Python 3.
2. Download or clone the project.
3. Open terminal in the project folder.
4. Run the following command:

```bash
python main.py
```
